java script이란 무엇인가?

자바스크립트는 기본적으로 객체(object)를 기반으로 하는 스크립트 언어이다. 웹을 개발할 때 내용은 HTML, 디자인은 CSS가 담당한다면 자바스크립트는 웹의 동작을 구현하는 역할을 한다. 자바스크립트는 웹 개발, 서버 측 프로그래밍, 모바일 앱 개발, 인공 지능 등 매우 다양한 분야에서 활용되고 있으며, 현재 대부분의 웹 브라우저들에는 자바스크립트 인터프리터가 내장되어 있기에, 매우 중요한 프로그래밍 언어라고 볼 수 있다.
